Vicalc

Keyboard First Diagram Editor

Features

  • Create diagrams only with your keyboard
  • Navigate with vim-like keyboard navigation
  • Export to .svg

Download

Current release is v0.2.1. Download an installer for your operating system

Windows .msi
Mac .dmg
Linux .AppImage | .deb*
